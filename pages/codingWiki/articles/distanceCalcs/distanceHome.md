# Distance Calculations

This section is dedicated to calulations used within programs and games.
Distance is a useful metric to use to detect collisions or find the closest object to another object.
Just by providing a two vector position can provide a useful distance measurement. 

These functions can be adapted to work in 2D and 3D enviroments.
